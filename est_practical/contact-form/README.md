# Contact Form — React

A clean contact form built with React and `useState`. Includes validation and displays submitted data below the form.

## Features

- Controlled inputs with `useState` for name, email, and message
- Client-side validation with per-field error messages
- Submitted data displayed below the form after successful submission
- Clear button to reset the displayed result

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
